Web application built with SAP Cloud Application Programming Model (CAP) for Instore Food Production.



### Key Capabilities

- **Central Recipe Management**: Create, manage, and transfer recipes directly to scales

- **Guided Production**: Step-by-step instructions on scales for producing pre-packaged fresh products

- **Accurate Tracking**: Gram-accurate recording of all production quantities and raw materials used

- **Automatic ERP Integration**: Direct connection between scales and SAP system for real-time data synchronization

- **Inventory Accuracy**: Ensures traceability of production and correct inventory management



## Tech Stack

- **Framework**: SAP CAP (Cloud Application Programming Model)

- **Database**: SAP HANA Cloud

- **Backend Integration**: SAP S/4HANA Public Cloud edition 2508 - Retail

- **Partner system**: Smart scales (Java application integration)

- **API**: REST

- **Runtime**: Node.js 

- **Language**: JavaScript/TypeScript

- **OData**: v4
